# Smart-Irrigation-System

Tech Stack: Python, Arduino, ESP8266, Soil Moisture Sensor, ThingSpeak
Description: An embedded system that monitors soil moisture levels and automatically turns on a water pump when the soil is dry. Data is sent to ThingSpeak for real-time monitoring and analysis.
Key Features:

Automated irrigation using soil moisture sensor

IoT-based data logging and visualization

Energy-efficient and low-cost design
